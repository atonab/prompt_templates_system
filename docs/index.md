## 🗂️ ¿Qué encontrarás aquí?

Utiliza el menú lateral para navegar por las diferentes áreas. La estructura está dividida por categorías estratégicas:

* **🎨 Imágenes y Diseño:** Prompts para generación visual, desde conceptos estilizados (como estética cyberpunk o elementos prehispánicos) hasta mockups de branding.
* **💻 Desarrollo y Sistemas:** Estructuras base para componentes web (React, Next.js, etc.), scripts de automatización y administración de servidores.
* **📈 Marketing y Agencia:** Copys para Google Ads, Meta Ads y estrategias de contenido estructuradas.

## 🚀 ¿Cómo usar las plantillas?

Cada prompt en esta Wiki sigue una estructura estandarizada para facilitar su uso:

1.  **Revisa las Variables:** Al inicio de cada documento verás una sección de `📋 Variables a preparar`. Reúne esa información antes de copiar el texto.
2.  **Copia el Prompt:** Usa el botón de copiado en la esquina superior derecha del bloque de código.
3.  **Reemplaza y Ejecuta:** Pega el texto en tu IA de preferencia (Claude, ChatGPT, Midjourney), reemplaza los valores entre corchetes `[ ]` y envíalo.

## 🔍 Búsqueda Inteligente

No necesitas saber dónde está guardado un archivo para encontrarlo:
* Usa la **barra de búsqueda** superior para buscar por palabras clave, el nombre del modelo (ej. *Claude 3.5*) o parte de la descripción.
* Haz clic en las **etiquetas (tags)** al final de las páginas para ver todos los prompts relacionados a un mismo tema.

## 💡 Consejos y Mejores Prácticas

* **Sé específico:** Cuanto más detallado sea tu prompt, mejores serán los resultados.
* **Usa ejemplos:** Proporciona ejemplos de lo que quieres para que la IA entienda mejor tus necesidades.
* **Itera:** No te conformes con el primer resultado. Experimenta con diferentes prompts hasta obtener lo que buscas.
* **Revisa y ajusta:** Cada modelo de IA es diferente, así que ajusta tus prompts según el modelo que estés utilizando.
